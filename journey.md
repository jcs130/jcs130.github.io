# 成长轨迹

## 方法论:兴趣驱动的闭环工程
我把兴趣当作起点,把闭环当作方法,把工程化与自动化当作结果:在真实系统中持续发现问题、拆解问题、解决问题,并将高频路径平台化、工具链化,最终走向自动化与智能化。

能力演进链条大致经历了:  
**数据库/数仓 → 数据采集/清洗/结构化 → 标注/特征 → 训练 → 评测/回归 → 工具链/平台 → QA 自动化 → AI 体系化。**

### 阿里巴巴:在成熟商业化大数据体系内锤炼生产级能力
- 在商业化数据环境中持续打磨数据工程、数据分析与数据产品体系能力,将"闭环工程"方法扩展到更大规模与更高稳定性约束下。

### 自动驾驶:从零开始,把既有能力迁移到更复杂的无人系统
